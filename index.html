Web Academy

Web Academy is a series of resources and workshops to guide new developers towards best practices on the web. See something out of place? Contribute and make it better!

Fundamentals of CSS

CSS (Cascading Style Sheets) is a language used to conduct the appearance and styling of HTML. This workshop covers everything you need to know about the basics of CSS.

Breaking Down the Box

At the core of every layout is the CSS box model, and this workshop will help you learn how to get the box model and positioning to work together.

Thinking Responsively

With a growing number of different devices creating flexible layouts is a necessity. Learn how the basics of creating fluid layouts and understanding the different concepts for thinking responsively.

CSS Preprocessors

CSS has very limited functionality and CSS preprocessors help change that. In this workshop you will learn about the different preprocessors available and how to use them to write cleaner css.

What's In Your Toolbox?

A craftsman is only as good as their tools, and if you have a limited toolbox you will not be as effecient as you should be. Learn about the various tools and debugging techniques on the web to help you work faster and more effeciently.

CSS3 Animations and Transitions

For years various javascript libraries have been used to create animations, but CSS is growing up with its own sets of tools for creating smooth animations and transitions for the web.
